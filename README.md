# Sales Data Analysis using Excel and Power BI

##  Project Overview
This project analyzes sales data to understand sales performance, trends, and key business insights using Excel and Power BI.

##  Data Files
- Raw sales data (Excel)
- Cleaned sales data (Excel)

##  Tools Used
- Microsoft Excel
- Power BI

##  Data Cleaning (Excel)
- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized column names
- Creating new correct column (correct sales = quantity*each price)
  
##  Data Analysis (Excel)
- Created pivot tables
- Analyzed monthly sales trends
- Compared product-wise performance

##  Pivot Table Screenshots
Screenshots of Excel pivot tables are included in this repository.

##  Power BI Dashboard
- KPI Cards (Total Sales, Total Orders)
- Sales trend analysis 
- Product and region-wise performance

##  Power BI Dashboard Screenshots
Power BI dashboard screenshots are included for easy viewing.

##  Key Insights
- Certain months show higher sales growth
- Few products contribute majority of revenue
- Clear sales trends help decision making

- Cleaned Excel file
